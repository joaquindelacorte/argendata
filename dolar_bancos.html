📊 Importar en Google Sheets

Usá la función IMPORTJSON o IMPORTDATA desde Google Sheets para que los datos se actualicen automáticamente desde esta página cada vez que abras la planilla.

Opción 1 — IMPORTDATA (CSV directo)

=IMPORTDATA("https://criptoya.com/api/bancostodos")

Opción 2 — Apps Script (recomendado, parsea JSON)

function importDolarBancos() { var url = "https://criptoya.com/api/bancostodos"; var response = UrlFetchApp.fetch(url); var data = JSON.parse(response.getContentText()); var sheet = SpreadsheetApp.getActiveSheet(); sheet.clearContents(); sheet.appendRow(["Banco", "Compra", "Venta", "Spread %", "Actualizado"]); var now = new Date().toLocaleString("es-AR"); for (var key in data) { var b = data[key]; var buy = b.totalask || b.ask || "-"; var sell = b.totalbid || b.bid || "-"; var spread = (buy && sell) ? (((sell - buy) / buy) * 100).toFixed(2) + "%" : "-"; sheet.appendRow([key, buy, sell, spread, now]); } }
  1. Abrí tu planilla de Google Sheets
  2. Menú Extensiones → Apps Script
  3. Pegá el código de la Opción 2 y guardá (Ctrl+S)
  4. Ejecutá la función importDolarBancos manualmente o configurá un trigger para actualización automática

📁 Exportar a Excel (XML)

Descargá un archivo .xml compatible con Excel que podés abrir directamente. Los datos reflejan la última cotización disponible. Para mantenerlo actualizado, volvé aquí y descargá nuevamente, o usá Power Query para conectarte directo a la API.

Power Query — conexión automática desde Excel

let url = "https://criptoya.com/api/bancostodos", source = Json.Document(Web.Contents(url)), toTable = Record.ToTable(source), expandCol = Table.ExpandRecordColumn( toTable, "Value", {"totalask","totalbid","ask","bid"}, {"Compra","Venta","ask_raw","bid_raw"} ), renameCol = Table.RenameColumns(toTable,{{"Name","Banco"}}), final = Table.SelectColumns(expandCol, {"Name","Compra","Venta"}) in final
  1. En Excel: Datos → Obtener datos → Desde otras fuentes → Consulta en blanco
  2. En el Editor de Power Query, menú Inicio → Editor avanzado
  3. Pegá el código M de arriba y presioná Listo
  4. Cerrá y cargá la consulta para ver los datos en la hoja
  5. Para actualizar: Datos → Actualizar todo
